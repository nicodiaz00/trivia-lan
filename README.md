# Trivia LAN 🎮

Plataforma para la gestión y juego de trivias corporativas.

## 🚀 Cómo Empezar

El proyecto está dividido en un frontend (React + Vite) y un backend ligero (JSON Server).

### 1. Instalación de Dependencias
```bash
cd frontend
npm install
```

### 2. Ejecución
**Terminal 1: Frontend**
```bash
cd frontend
npm run dev
```

**Terminal 2: Backend**
```bash
cd frontend
npm run server
```
