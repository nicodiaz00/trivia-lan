# Trivia LAN 🎮

## 🚀 Cómo Empezar

El proyecto está dividido en un frontend (React + Vite) y un backend ligero (JSON Server).

### 1. Instalación de Dependencias
```bash
cd frontend
npm install
```
Espera a que termine de instalar todo.

### 2. Ejecución
**Terminal 1: Frontend**
```bash
npm run dev
```
Espera a que termine de compilar. Una vez compilado debes tener el backend corriendo para que la aplicación funcione. Para correr el backend ejecuta el siguiente comando:

**Terminal 2: Backend**
```bash
cd frontend
npm run server
```
Espera a que termine de compilar. Una vez compilado, se abrirá una pestaña en tu navegador con la aplicación en la dirección: http://localhost:3000
